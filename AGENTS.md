# AGENTS.md

Инструкции для Codex при работе с проектом Gym Tracker.

---

## Общение и стиль

Отвечай на русском, коротко и по делу. Пользователь — vibecoder: не читает код, оценивает результат **визуально** через браузер и UI.

- Визуальные изменения — показывать в браузере или скриншотом.
- Изменения "под капотом" — объяснять коротким блоком:
  ```
  ## Что изменилось
  🖥 UI  — [что видно]
  ⚙️ API — [логика/endpoint]
  ```

## Правила кодирования

1. **Минимум кода.** Только то, что решает задачу. Без спекулятивных фич и абстракций про запас.
2. **Хирургические изменения.** Трогать только нужное. Не рефакторить соседний код без запроса.
3. **Думай перед кодом.** Если задача неоднозначна — уточнить, не делать молчаливых допущений.
4. **Критерии успеха.** Перед реализацией — определить как проверим результат.

## Текущий редизайн → Expo/RN (22.07.2026)

Проект переезжает с Vite-веб-скелета на **Expo / React Native** для публикации в App Store
(личное приложение, без логина/мультипользователя). Дизайн готов и лежит в
`design/claude-design-export/` — это handoff-бандл из Claude Design (Fable 5), прочитай
`design/claude-design-export/README.md` первым, там инструкция для кодинг-агента.

**Убрать при переносе:** все AI-фичи (Gemini pre-workout совет, weekly-отчёт, чат-ассистент) —
были демо-фичей AI Studio прототипа, в новом дизайне их нет.

**Новая механика (заложена в дизайн):**
- **Rest** — таймер между подходами (как сейчас), звук в конце = пора начинать подход.
- **Pause** — отдельная кнопка, доступна в любой момент тренировки, замораживает счётчик
  активного времени (не искажает итоговое время). Summary показывает разбивку: общее время /
  активное / кол-во и длительность пауз.
- **Skip exercise** (объём не считается) vs **End exercise early** (сделанные подходы всё равно
  в объём) — раздельные действия, фикс старого бага.
- Необязательные теги причины (Устал / Отвлёкся / Не хватило времени / Дискомфорт) при паузе
  или раннем завершении — просто метка в данных.

**Новый раздел «Добавки»:** чек-лист приёма по времени дня, склад/остатки с warning при низком
запасе, стрик/прогресс адгеренса, редактор расписания.

**Навигация:** 5 табов — Главная · Тренировки · Добавки · История · Настройки.
